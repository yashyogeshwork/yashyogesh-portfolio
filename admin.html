Site Editor

Edit any text on the site and manage the sketch wall — no coding needed.

How publishing works (3 steps):
  1. Edit anything below, then press Download updated content.js.
  2. Go to your GitHub repo → open the js folder → click content.js → click the pencil/edit — or simpler: use Add file → Upload files into the js folder and upload your downloaded file (it replaces the old one).
  3. Commit. Netlify republishes automatically in about a minute. Done.

Adding / replacing sketches:
  1. On GitHub, open the images/sketches folder → Add file → Upload files → upload your sketch image (JPG or PNG).
  2. To replace an existing sketch: upload a new file with the same filename — nothing else needed, it's live after commit.
  3. To add a new sketch: upload the file, then add its filename in the Sketch Wall section below and download/upload the new content.js.
